# Shell Permissions #
<pre>
1)  0-iam_betty                   -          Switch the current user to Betty
2)  1-who_am_i                    -          Prints the effective name of the current user.
3)  2-groups                      -          Prints all the groups the current user is part of.
4)  3-new_owner                   -          Changes the owner of the file to Betty.
5)  4-empty                       -          Creates an empty file.
6)  5-execute                     -          Add execute permissions to the owner.
7)  6-multiple_permissions        -          Adds execute permission to the owner and the group owner, and read permission to other users, to the file hello.
8)  7-everybody                   -          Add execute permission to everyone.
9)  8-James_Bond                  -          Add all permissions to other users and remove all permissions for the owner and group.
10) 9-John_Doe                    -          Set the file permission to a certain set permissions.
11) 10-mirror_permissions         -          Reference the permissions of the olleh file to set the file permissions for hello file.
12) 11-directories_permissions    -          Add execute permissions to the subdirectories of the current directories(files are not to be changed.
13) 12-directory_permissions      -          Create a directory with 751 permissions.
14) 13-change_group               -          Change the group owner of the file hello.
15) 100-change_owner_and_group    -          Change the owner and the group owner for files and directories in the current directory.
16) 101-symbolic_link_permissions -          Change the owner and the group owner for a symbolic link.
17) 102-if_only                   -          Change the owner of hello if it only belongs to guillaume.
18) 103-Star_Wars                 -          Play StarWars IV episode in the terminal.
</pre>
